Preparing an .stl file for the 3D printer

1. Open the Bambu studio app

2. Click File-Import-stl files

3. Select the correct extruder nozzle size

4. Select the correct filament used

5. Move tool: to move the model around om the build plate

6. Rotate: this tool is used to change the angle or position of the model

7. Resize: There's even a scale for specific size changes

8. Select support if needed

9. Generate preview

10. Once satisfied export the model to your USB drive

11. Copy to USB

12. Insert the drive into the 3D printer and select model name in the file

13. Print! Good luck
